# Hazelrigg Enterprises Website

Premium aircraft and industrial equipment showcase built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
npm run build
```

## Project Structure

- `src/app/` - Next.js App Router pages
- `src/components/` - React components
- `src/data/` - Aircraft data
- `src/types/` - TypeScript definitions

## Features

- Premium aviation-themed design (charcoal/silver/gold palette)
- Aircraft listing with category/manufacturer filters
- Individual aircraft product pages with gallery and enquiry forms
- Responsive navigation with dropdown menus
- Mobile-first Tailwind CSS styling

## Deployment

Deploy to Vercel via GitHub integration.
